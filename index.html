使用說明 / Help

📋 基本操作
新增任務點擊工具列的 + 新增任務
編輯任務點擊甘特圖中的任務橫條，開啟編輯彈窗
刪除任務開啟編輯彈窗後，點擊左下角的 刪除 按鈕
專案名稱點擊左上角的專案名稱即可直接編輯
🖱️ 拖曳操作
移動任務拖曳計劃條（外框）或實際條（實色）的中央左右移動
調整日期拖曳橫條的左側邊緣調整開始日，右側邊緣調整結束日
吸附日期自動吸附至整天
🗂️ 子任務與視圖模式
新增子任務開啟父任務的編輯彈窗，在「子任務」區塊點擊 + 新增子任務
展開／收合（層級模式）點擊父任務橫條，子任務縮排展開於下方
子面板模式切換至 ⊞ 分頁 後，點擊父任務，在下方展開獨立子甘特圖
💾 CSV 檔案
開啟點擊 開啟 CSV，記錄檔案位置以便直接回存
儲存點擊 儲存 或按 CtrlS，寫回目前開啟的檔案
另存點擊 匯出 另存新的 CSV 副本，不影響目前開啟的檔案
未儲存提示有未儲存變更時，檔案名稱顯示變成琥珀色並出現 ●
🖼️ 匯出圖片
下載 PNG點擊 匯出圖片，儲存高解析度 PNG（2× 縮放）
複製到剪貼簿點擊 複製圖片，可直接貼至 Word、Slides 等文件（需 Chrome 84+）
📤 匯出 HTML 預覽
匯出 HTML點擊 匯出 HTML，生成單一可攜帶的 .html 檔案，任何人用瀏覽器開啟即可查看甘特圖並展開子任務階層
注意需在 VS Code Live Preview 或 GitHub Pages 下使用，不支援直接雙擊 file:// 開啟
🌐 語言切換
切換語言點擊工具列右側的 中 / EN 按鈕，介面即時切換中英文，設定儲存於瀏覽器
📋 Basic Operations
Add taskClick + Add Task in the toolbar
Edit taskClick any task bar to open the edit dialog
Delete taskOpen edit dialog → click Delete (bottom-left)
Project nameClick the project name (top-left) to edit inline
🖱️ Drag & Drop
Move taskDrag the center of a bar (plan or actual) left / right
ResizeDrag the left edge to change start; right edge to change end
SnappingDates snap to whole days automatically
🗂️ Sub-tasks & View Modes
Add sub-taskOpen a parent task → click + Add Sub-task
Expand / CollapseClick a parent task bar to toggle its sub-tasks (Hierarchy mode)
Sub-panel modeSwitch to ⊞ Sub-panel, click a parent to open sub-chart below
💾 CSV Files
OpenClick Open CSV — remembers the file for in-place saving
SaveSave button or CtrlS — writes back to the open file
ExportDownloads a new CSV copy; does not update the open file
Unsaved indicatorFilename turns amber with a dot (●) when changes are unsaved
🖼️ Export Image
Download PNGClick Export Image — saves a 2× resolution PNG
Copy to clipboardClick Copy Image — paste into Word, Slides, etc. (Chrome 84+)
📤 Export HTML Preview
Export HTMLClick Export HTML to generate a self-contained .html file — anyone can open it in a browser to view the chart and expand the task hierarchy
NoteRequires VS Code Live Preview or GitHub Pages; does not work when opened directly as file://
🌐 Language
Switch languageClick 中 / EN in the toolbar to toggle the UI language instantly; preference is saved in the browser
